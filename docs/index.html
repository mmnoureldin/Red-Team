4.a Domain Escalation - Zerologon

post @ 2023-05-01

Introduction

Due to the flawed implementation of the cryptographic authentication scheme used by Netlogon Remote Protocol (MS-NRPC) that causes authentication to be bypassed, attackers can reset the domain controller machine password.

This vulnerability has been assigned CVE-2020-1472.

Let's reset the DC password. Now, we can just use the DC machine hash to perform DCSync attack, Impacket `secretdump.py` can be used through socks proxy to achieve this. Note that, this will leave the domain unfunctioning unless the DC password has been reset. ## Observation As expected the DC machine account password has been reset as in the event id `4742`, moreover the action was launched from our initial access machine `CASTELBLACK`. Furthermore, there are a bunch of NTLM authetication attempts using the new DC machine password hash after the password reset, this may be due to the authentication trials that were performed to ensure the exploitation success and dumping the domain hashes. {%asset_img image-20230501004404863.png" target="">`sharpzerologon`
